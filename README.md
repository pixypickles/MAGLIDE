# Floating Magic Fighter

常時浮遊型の軽量2D対戦アクションゲーム試作です。  
スマートフォンのブラウザで動作し、GitHub Pagesへそのまま公開できます。

## 特徴

- 常時浮遊。歩行アニメーションなし
- 円形バーチャルスティック
- A: オートターゲット魔力弾
- B: 魔力キック
- C: 4方向の専用軌道による高速移動
- G: ガード／ジャストガード反射
- 敵は色違いで同性能
- Canvas描画中心で画像枚数を抑えた軽量構成

## 操作

### スマートフォン

- 左下スティック: 移動・技方向
- A: 魔力弾
- B: 魔力キック
- C: 高速移動
- G: ガード

### PC確認用

- 移動: 矢印キー / WASD
- A: J
- B: K
- C: L
- G: I

## 起動

ローカルでは単純なHTTPサーバーで起動してください。

```bash
python -m http.server 8000
```

ブラウザで `http://localhost:8000` を開きます。

## GitHub Pages

1. このフォルダをGitHubリポジトリへアップロード
2. Settings → Pages
3. Branchを `main`、Folderを `/root` に設定
4. 表示されたURLをスマートフォンで開く

## 主な調整場所

- `src/config.js`: ダメージ、速度、受付時間など
- `src/game.js`: ゲーム処理
- `src/render.js`: キャラクターとエフェクト描画
- `docs/spec.md`: ゲーム仕様
- `assets/character-data.json`: キャラクター定義

## 軽量化方針

キャラクター本体はCanvasの少ない線で描画しています。  
将来スプライトへ置き換える場合も、32×32〜48×48程度、各行動1〜3枚を想定しています。

# Floating Magic Fighter v3

GitHub Pages向けの1ファイル版です。

## 公開方法

リポジトリ直下に、この `index.html` を置いてください。
以前の `src` フォルダや古い `index.html` は削除してください。

GitHub Pagesを更新後、スマートフォン側でページを再読み込みしてください。
古い表示が残る場合はURL末尾に `?v=3` を付けて開いてください。

例:

`https://ユーザー名.github.io/リポジトリ名/?v=3`

## ボタン

- 上: A 魔力弾
- 左: G ガード
- 右: B キック
- 下: C 移動技

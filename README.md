# Floating Magic Fighter v7

## C移動技の軌道を4方向で個別化

- 前: 波打ちながら高速前進
- 後: 後退しながら1回転するループ軌道
- 上: 一度上昇 → 斜め下降 → 再上昇
- 下: 一度下降 → 斜め上昇 → 再下降
- キャラクターと球状バリアの傾きが、軌道の接線方向へ追従
- 全軌道で青いバリアに接触判定あり

# Floating Magic Fighter v8

## C移動技の軌道

- 前C: 前方へ波打ちながら高速前進
- 上C: 前方斜め上へ進みながら小さく揺れる
- 下C: 前方斜め下へ進みながら小さく揺れる
- 後C: 最初に少し前へ出て、大きく一周し、開始位置付近へ戻る
- 後Cの終了位置は開始地点より約25px後方

すべて青い球状バリアをまとい、接触判定があります。

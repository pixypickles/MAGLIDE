# Floating Magic Fighter v5

## 更新内容

- 通常移動速度をさらに高速化
- 大ダメージで即座に落下しないよう変更
- 大ダメージ時の流れ:
  1. 攻撃方向へ横に吹き飛ぶ
  2. 画面端の壁へ衝突
  3. 衝突エフェクトと短い硬直
  4. 画面下へ落下
  5. 下からすぐ浮上
- 軽い攻撃は従来どおり、横ノックバックと下降

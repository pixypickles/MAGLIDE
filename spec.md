# ゲーム仕様

## 基本

- 2D横視点
- 地面は表示しない
- 両者とも常時浮遊
- 敵は色違いで同性能
- 1対1を基本とする

## 浮遊と被弾

- 待機中は片膝を軽く曲げた浮遊姿勢
- 小被弾は数ピクセル下がる
- 強い一撃は画面下へ消える
- 消えた直後に下から浮上
- 意識を失うと浮遊魔法が切れる設定

## A: 魔力弾

- ダメージ小
- 常時オートターゲット
- 入力方向に対応した位置から生成し、その後は敵へ向けて進む
- 上: 頭上側から発射
- 下: 足元側から発射
- 前: 初速が速く、通常速度へ減速
- 後: 初速が遅く、通常速度へ加速
- 連射可能

## B: 魔力キック

- ダメージ大
- 物理ダメージ + 魔法ダメージ
- 敵が右なら右向き、左なら左向き
- 足を伸長させない
- 下から頭の高さまで蹴り上げる
- 足先だけ発光
- 残像は三日月形
- 魔力を飛ばさず、足が直接接触した時のみヒット

## C: 高速移動

入力した4方向ごとに固有軌道を使用します。

- 上: 上方へ弧を描く
- 下: 下方へ大きく沈んで抜ける
- 前: 小さく波打ちながら高速前進
- 後: 後方へ切り返す独特の軌道
- バリアをまとって接触すると中ダメージ
- 飛行機の旋回や宙返りに近い爽快感を重視

## G: ガード

- 通常ガード: ダメージ軽減
- ジャストガード: 相手の魔力弾を反射
- 受付時間は短めだが、スマートフォン向けに極端に厳しくしない
- 移動技との複合入力は初期版では採用せず、C単独で体当たり判定を持たせる

## 操作設計

複合コマンドは入力失敗によるストレスを避けるため、初期版では使用しません。

- 円形スティック
- A 魔力弾
- B キック
- C 移動技
- G ガード
